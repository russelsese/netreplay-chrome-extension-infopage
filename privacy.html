Privacy Policy

Effective date: April 18, 2026

NetReplay does not collect, transmit, or store any of your personal data or browsing activity. Everything stays on your device.

Overview

NetReplay ("the Extension") is a Chrome extension that captures and replays network requests made by web pages you visit. This privacy policy explains what data the Extension accesses, how it is used, and what it does not do.

Data the Extension Accesses

To provide its functionality, the Extension reads the following data locally on your device only:

  • Network request URLs, methods, headers, and request bodies initiated by the active tab
  • Network response status codes, headers, and response bodies
  • Extension UI state (e.g., whether capture is active) stored via Chrome's storage API

Data the Extension Does NOT Collect

The Extension never:

  • Sends any data to external servers or third parties
  • Stores captured request data beyond your current browser session
  • Tracks your browsing history or page visits
  • Collects personal information such as names, email addresses, or identifiers
  • Uses analytics, crash reporting, or advertising services

Permissions Used

The Extension requests the following Chrome permissions:

  • activeTab — to inject the capture script into the current tab when you click the icon
  • storage — to persist UI state (e.g., panel open/closed) across page navigations

No other permissions are requested. The Extension does not use broad host permissions.

Data Security

Because all data remains local to your browser, no server-side security measures are required. Captured request data (including any credentials in headers or bodies) is held in memory only while the panel is active and is discarded when you close the tab or navigate away.

Children's Privacy

The Extension is a developer tool not directed at children under the age of 13 and does not knowingly collect any personal information from children.

Changes to This Policy

If this policy changes, the updated version will be published at this URL with a new effective date. Continued use of the Extension after changes constitutes acceptance of the updated policy.
