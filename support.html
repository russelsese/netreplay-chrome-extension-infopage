Common Questions

Why aren't requests being captured?
Make sure you click the Capture button before interacting with the page. Requests made before clicking Capture are not recorded.

The replay returned a different response — why?
Replayed requests run fresh against the server. Session tokens, cookies, or server state may have changed since the original request.

Can I capture WebSocket messages?
Version 1.0 captures fetch and XMLHttpRequest only. WebSocket support is planned for a future release.
